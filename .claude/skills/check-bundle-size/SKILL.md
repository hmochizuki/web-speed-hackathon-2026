# バンドルサイズチェック

ビルドを実行してバンドルサイズを計測し、前回の結果と比較して改善・悪化を報告する。

## 手順

Agent ツールで `check-bundle-size` subagent を起動する。
- subagent_type は指定しない（general-purpose）
- プロンプト: 「バンドルサイズをチェックしてください」

引数が指定された場合:
- `--build-only`: ビルドのみ実行し、ベースラインとして保存する（比較しない）
- `--compare-only`: ビルドせず、既存の最新結果とベースラインを比較する
- `--baseline <file>`: 比較元のベースラインファイルを指定（ファイル名のみ or `application/temp/` 内のパス）。未指定時はユーザーに選択を求める
- `--baseline list`: 利用可能なベースラインファイル一覧を表示して終了
- 引数なし: ビルド → 比較 の両方を実行する

`--baseline` が指定された場合、プロンプトに「ベースライン: <file>」を含めて渡す。
`--baseline list` の場合、プロンプトは「利用可能なベースラインファイルを一覧表示してください」とする。

subagent の結果をそのままユーザーに表示する。
